METAR Report decoding web service

API Reference

  • Get health check

      GET /metar/ping

    Returns {'data': 'pong'}

  • Get weather info

      GET /metar/info/?scode=${scode}&nocache=${nocache}
    Parameter Type Description
    scode string Required. 4 character station code of the weather station we intend to get the METAR report from
    nocache string Optional. Set using 1/yes/true/T if the user wishes to get fresh data
